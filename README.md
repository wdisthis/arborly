# arborly
A static web tool to transform indented text outlines  into beautiful tree diagrams. exportable to pdf and png

#### Repo structure:
```text
arborly/
├── index.html
├── styles/
│   └── main.css
├── js/
│   ├── parser.js (to precessing inndentation)
│   ├── renderer.js (to draw tree)
│   └── exporter.js (PDF/PNG/SVG)
└── README.md
```
